終止使用某Port的程式

查詢佔用的Process其PID為何

$ lsof -i:9090

COMMAND   PID    USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
java    29313 AlanJui  189u  IPv6 0xd5c365aad9209f05      0t0  TCP *:websm (LISTEN)


以PID終止Process

$ sudo kill -9 29313

驗證Process己被終止

$ lsof -i:9090
